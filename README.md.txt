LoginApp

LoginApp es una aplicación de escritorio escrita en Java que permite gestionar usuarios mediante una interfaz gráfica. Incluye funcionalidades de inicio de sesión, registro de nuevos usuarios, actualización y eliminación de usuarios. Utiliza patrones de diseño como Singleton y Factory para garantizar un diseño escalable y mantenible.

Características

Inicio de sesión seguro (validación de usuario y contraseña).
Registro de nuevos usuarios con validación de campos.
Gestión de usuarios:
Visualización de usuarios registrados.
Actualización de contraseñas.
Eliminación de usuarios.
Interfaz gráfica amigable utilizando Swing.
Uso de patrones de diseño como Singleton y Factory.

Uso

Al iniciar la aplicación, se muestra la ventana de inicio de sesión.
Si no tienes una cuenta, puedes registrarte haciendo clic en el botón Registrarse.
Una vez registrado, usa tu nombre de usuario y contraseña para iniciar sesión.
Después del inicio de sesión exitoso:
Podrás ver la lista de usuarios registrados.
Puedes actualizar la contraseña de un usuario seleccionado.
Puedes eliminar usuarios (excepto el administrador predeterminado).
Para cerrar sesión, usa el botón Cerrar Sesión.
